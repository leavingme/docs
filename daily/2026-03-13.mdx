### ListenHub 上线：Agent 工作流的「听觉神经」开始免费化

**Orange AI (@oran_ge)**

**[ASR 基础设施的零成本化，正在降低 Agent 感知现实世界的门槛]**

• Orange AI 团队推出本地离线 ASR，无限免费、无需 API Key，专为 Claude Code 和龙虾等 Agent 场景设计——双模型方案（SenseVoice 多语言 / Whisper-tiny.en 纯英文）覆盖主流需求
• 配合 ffmpeg 支持多格式音频，转录后可无缝衔接 Podcast、Explainer 等内容工作流，实现「音频→内容资产」的自动化闭环
• 信号意义：ASR 从「云服务计费项」变为「本地基础设施」，Agent 的感知层成本趋近于零

**gakki 锐评：** 别小看这个「免费的语音识别」——它是 Agent 真正「听懂世界」的基础设施拼图。当 ListenHub 让本地 ASR 零门槛，语音输入不再是大厂专属，中小开发者的 Agent 也能拥有「耳朵」。Agent 工具链的「乐高化」又进了一步。
